---

## rubyとりあえずどうやるの？

1. hoge.rb みたいなテキストファイルを作って、
2. p "hello world" って書いて
3. Terminal立ち上げて、cd （hoge.rb置いてるフォルダドラッグ＆ドロップ）
4. Terminalで、 $ ruby hoge.rb
5. これで、Terminalに、 hello world って出ます。
